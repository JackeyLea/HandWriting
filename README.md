# HandWriting

Qt/C++/OpenCV手写字符识别

包括svm/knn/pytorch识别

本仓库包括两部分，一个是训练测试mnist，一个是手写字符和识别。

## Base

Manjaro Linux 18.0.3

Qt 5.13.1

GCC 9.3.0

OpenCV 4.2.0

https://blog.csdn.net/qq_26056015/article/details/100176334

这是我编译的，也是我现在用的

直接用Qt编译就可以了，注意库文件位置

MNIST数据集与可执行文件处于同一位置
