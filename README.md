# HandWriting

Qt/C++/OpenCV/SVM/KNN手写字符识别

本仓库包括两部分，一个是训练测试mnist，一个是手写字符和识别。

依赖Qt+OpenCV，直接用Qt编译就可以了，注意库文件的位置。

代码解析，源码使用在：<a href="https://feater.top/series/opencv/2223/">Qt+OpenCV+SVM+KNN的MNIST手写字符识别</a>

## 手写字符识别

### Function

1、训练mnist数据集

2、测试mnist数据集训练结果

3、在界面用鼠标画字符

4、点击Go按钮进行识别

5、界面弹出识别结果
